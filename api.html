API Reference

Integrate the Sovereign-Village-OS Identity Registry into your enterprise applications.

GET /v1/identity/verify

Retrieve the verified status and Pillar Tier of a registered individual using their Sovereign ID.

// Example Request
curl -X GET "https://api.tgsobausa.org/v1/identity/verify?id=PHL-8821-SOV" \
    -H "Authorization: Bearer YOUR_API_KEY"
// Example JSON Response
{
  "id": "PHL-8821-SOV",
  "status": "VERIFIED",
  "tier": "Architect",
  "goodness_score": 942,
  "last_checkin": "2026-03-14T10:00:00Z"
}
